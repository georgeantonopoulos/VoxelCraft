# Tech Stack

## Core Technologies

*   **Frontend Framework:** React 19 + TypeScript
*   **3D Graphics:** Three.js + React Three Fiber
*   **Physics Engine:** Rapier (via @react-three/rapier)
*   **State Management:** Zustand
*   **UI Debugging/Controls:** Leva
*   **Build Tool:** Vite
*   **Styling:** Tailwind CSS
*   **Testing Framework:** Vitest
*   **Local Database:** Dexie (for IndexedDB operations)

## Key Libraries & Tools

*   `@react-three/drei`: A collection of useful helpers and abstractions for React-three-fiber.
*   `@react-three/postprocessing`: Post-processing effects for React-three-fiber.
*   `fast-simplex-noise`: For procedural terrain generation.
*   `n8ao`: For screen-space ambient occlusion.
*   `three-custom-shader-material`: For advanced custom shader development.
*   `typescript`: For type-safe JavaScript.
*   `postcss`, `autoprefixer`: For CSS processing.

## Development Environment

*   **Runtime:** Node.js (v18 or higher recommended)
*   **Package Manager:** npm (or yarn)
